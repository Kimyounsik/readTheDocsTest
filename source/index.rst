HSP SDK example
==================================

Hangame Smart Platform SDK
 
HSP SDK는 NHN Entertainment의 탄탄한 게임플랫폼 기술력을 기반으로
쉽고 편하게 NHN Entertainment 의 플랫폼을 이용하여 게임을 개발할 수 있도록 돕는 강력한 도구입니다.
 
HSP SDK는  사람, 기술, 프로세스의 조화가 빛나는
No. 1 NHN Entertainment의 경쟁력입니다.
 
HSP SDK를 처음 접하신다면 HSP SDK Quick Guide로 시작하세요!
 
해당 브라우저는 비디오 태그를 지원하지 않습니다. IE9이상, 크롬, 파이어폭스 또는 사파리를 이용해주세요.
